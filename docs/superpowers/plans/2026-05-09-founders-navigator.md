# Founder's Navigator Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build and deploy a 4-screen interactive prototype that takes a founder's scenario, extracts a structured profile via Claude, scores 213 official Utah resources, and presents a ranked action plan with Markdown/JSON export.

**Architecture:** Greenfield Next.js 15 App Router project. Content lives in `data/` (catalog + config) and is never hardcoded in components. Scoring is a pure client-side function — no AI on the plan screen. The only AI call is `POST /api/extract` which returns a `FounderProfile`.

**Tech Stack:** Next.js 15 App Router, TypeScript, Tailwind CSS v3, Anthropic SDK (`@anthropic-ai/sdk`), Vitest + @testing-library/react

```ts
  description: row.description,
  communities: row.Communities ? row.Communities.split('|').map((s: string) => s.trim()).filter(Boolean) : [],
  industries: row.Industries ? row.Industries.split('|').map((s: string) => s.trim()).filter(Boolean) : [],
  locations: row.Locations ? row.Locations.split('|').map((s: string) => s.trim()).filter(Boolean) : [],
  topics: row.Topics ? row.Topics.split('|').map((s: string) => s.trim()).filter(Boolean) : [],
  link: row.link || '',
  email: row.email || '',
}))

writeFileSync('data/catalog.json', JSON.stringify(catalog, null, 2))
console.log(`Wrote ${catalog.length} entries to data/catalog.json`)
```

```ts
import type { CatalogItem, FounderProfile, Recommendation } from './types'

export function fitLabel(score: number): 'Strong match' | 'Good match' | 'Worth knowing' {
  if (score >= 4.0) return 'Strong match'
  if (score >= 2.5) return 'Good match'
  return 'Worth knowing'
}

export function score(catalog: CatalogItem[], profile: FounderProfile): Recommendation[] {
  const isUrgent = /week|urgent|immediate|asap/i.test(profile.urgency)

  const scored = catalog.map((item) => {
    let raw = 0
    const matched: string[] = []

    if (item.topics.includes(profile.stage)) {
      raw += 1
      matched.push(`aligns with your goal to ${profile.stage}`)
    }
    if (item.industries.includes(profile.industry)) {
      raw += 1
      matched.push(`serves ${profile.industry} businesses`)
    }
    if (item.locations.includes(profile.location)) {
      raw += 1
      matched.push(`available in ${profile.location} County`)
    }
    if (item.communities.includes(profile.community) || item.communities.includes('Any')) {
      raw += 1
      if (profile.community !== 'Any' && item.communities.includes(profile.community)) {
        matched.push(`is specifically designed for ${profile.community} founders`)
      }
    }
    if (isUrgent && item.topics.includes('Start a Business')) {
      raw += 0.5
    }

    const fitScore = Math.min(5, (raw / 4.5) * 5)
    const matchedCriteria = matched.length
    const whyItFits = matched.length > 0
      ? `This resource ${matched.join(', ')}.`
      : 'This resource may be relevant to your situation.'

    return {
      ...item,
      fitScore,
      fitLabel: fitLabel(fitScore),
      matchedCriteria,
      whyItFits,
    }
  })

  return scored
    .sort((a, b) => b.fitScore - a.fitScore)
    .slice(0, 5)
    .map((item, i) => ({ ...item, rank: i + 1 }))
}
```

```ts
export function toMarkdown(recs: Recommendation[], profile: FounderProfile): string {
  const today = new Date().toISOString().split('T')[0]
  const frontmatter = [
    '---',
    `generated: ${today}`,
```

```ts
Return ONLY a valid JSON object. No markdown, no explanation.

Fields and accepted values:

- location: a Utah county name. Choose from: Beaver, Box Elder, Cache, Carbon, Daggett, Davis, Duchesne, Emery, Garfield, Grand, Iron, Juab, Kane, Millard, Morgan, Piute, Rich, Salt Lake, San Juan, Sanpete, Sevier, Summit, Tooele, Uintah, Utah, Wasatch, Washington, Wayne, Weber. - community: one of exactly: "Any" | "Women" | "Veteran" | "Rural" | "Student" | "Multicultural" | "New American"
- primaryGoal: short phrase describing the founder's main goal
- urgency: short phrase describing timeline or urgency
- supportPreference: one of "self-service" | "human" | "mix"
- constraints: short phrase about limitations or constraints, or empty string
- assumptions: array of field name strings that you inferred rather than directly read from the scenario

  "scenario": "I'm a veteran starting a manufacturing business in Ogden.",
  "location": "Weber",
  "stage": "Start a Business",
  "industry": "Manufacturing",
  "community": "Veteran",
  "primaryGoal": "Register business and find startup resources",
  "urgency": "Ready to start now",
  "supportPreference": "human",
  "constraints": "",
  "assumptions": ["supportPreference"]
}`

export async function POST(request: Request) {
  try {
    const { scenario } = await request.json()
    if (!scenario || typeof scenario !== 'string') {
      return NextResponse.json({ error: 'scenario required' }, { status: 400 })
    }

    const message = await client.messages.create({
      model: 'claude-sonnet-4-6',
      max_tokens: 1024,
      system: SYSTEM_PROMPT,
      messages: [{ role: 'user', content: scenario }],
    })

    const text = message.content[0].type === 'text' ? message.content[0].text : ''
    const profile: FounderProfile = JSON.parse(text)
```
